# IdeaForge CLI

MVP Sistema de Debate de Ideias Assistido por IA. Permite que desenvolvedores transformem ideias vagas em planos estruturados através de debate entre agentes.

## Requisitos
- Python 3.10+
- Ollama (opcional, para execução local)

## Instalação
```bash
pip install -r requirements.txt
```

## Execução
```bash
python src/cli/main.py
```
